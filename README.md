
מערכת המאפייה נועדה לנהל בצורה ממוחשבת את תהליך ההזמנות במאפייה.
המערכת מאפשרת ניהול לקוחות, עובדים, מוצרים והזמנות, תוך שמירה על קשרים לוגיים בין כלל הנתונים.

נתונים ופונקציונליות עיקרית-

המערכת מנהלת את הנתונים הבאים:
•	לקוחות ופרטי ההתקשרות שלהם
•	עובדים ותפקידיהם
•	קטגוריות מוצרים
•	מוצרים, מחירים וזמינות
•	הזמנות שבוצעו במאפייה
•	פירוט פריטי ההזמנה (כמות ומחיר)
המערכת מאפשרת יצירה ומעקב אחר הזמנות ושליפת נתונים לצורכי ניהול.

עיצוב בסיס הנתונים-

בסיס הנתונים עוצב באמצעות תרשימי ERD ו־DSD בתוכנת .ERDPlus
הסכמה מנורמלת לרמה של 3NF, וכוללת שימוש בשדות תאריך כגון תאריך הרשמה ותאריך הזמנה.

אכלוס נתונים וגיבוי-

הנתונים הוזנו לבסיס הנתונים באמצעות שלוש שיטות:
•	ייבוא מקבצי Excel
•	פקודות INSERT בקובץ SQL באמצעות סקריפט בפייתון 
•	שימוש באתר Mockaroo
בכל טבלה הוזנו לפחות 400 רשומות.
בוצע גיבוי מלא של בסיס הנתונים ונבדקה תקינות השחזור.
